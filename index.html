Connect to Gateway

// remote.it → find gateway → paste endpoint → get powershell command
1
Open Remote.it and search for your gateway
Go to app.remote.it/#/devices and search for your gateway name.
Open Remote.it ↗
2
Click the SSH service and connect
In the device list, click your gateway. Find the SSH service and click Connect. Remote.it will generate a public endpoint.
3
Copy the public endpoint and paste it here
The endpoint looks like proxy16.rt3.io:37081 — copy it from Remote.it and paste below.
4
Your PowerShell command
Copy this command, open PowerShell, paste it, and press Enter. Enter the password when prompted.
PS>ssh -p <port> binmaster@<host>
⚠️
Endpoints are time-limited. If the connection drops or the endpoint expires, return to Remote.it, click the SSH service again, and paste the new endpoint above.
ℹ️
The Online Status in Remote.it reflects a background ping. If it shows online, the gateway is reachable.

Run Gateway Commands

// once connected via ssh — use these commands on the gateway

CommandPurposeNotes
bcsensors View sensor readings Also sends manual readings to BinCloud
sudo reboot Reboot the gateway Only possible via SSH
uptime How long gateway has been running Good first diagnostic

Troubleshoot Offline Gateway

// when the gateway is not connected to remote.it
⚠️
If the gateway is not connected to Remote.it, SSH access is impossible until connectivity is restored.
POSSIBLE CAUSES
#CauseAction
1Gateway is powered downCheck physical power / PDU / breaker
2Network cable / WiFi disconnectedInspect Ethernet connection on the gateway
3Customer firewall blocking outbound trafficAsk IT to whitelist outbound TCP for Remote.it
4Hardware damage to gateway unitMay need a replacement unit shipped
DIAGNOSTIC CHECKLIST
  • Confirm gateway shows OFFLINE in Remote.it — not just slow to respond
  • Ask on-site contact to check power LEDs on the gateway unit
  • Ask on-site contact to re-seat or replace the Ethernet cable
  • Check whether the customer recently changed firewall or router rules
  • If all above checks pass — escalate for possible hardware replacement
